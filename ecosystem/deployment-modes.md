# Deployment Modes

| Mode | Purpose |
|---|---|
| Local Desktop | single-user full AI |
| Home Lab | persistent personal infrastructure |
| Family Office | private institutional continuity |
| PMA Deployment | member operations and memory |
| Team Server | collaborative enterprise usage |
| Air-Gapped | isolated high-security deployment |
| Edge Mesh | distributed edge AI systems |
| Hybrid Cloud | selective external reasoning |
