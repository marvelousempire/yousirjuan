# Hardware SKUs

**Tagline:** From $150 to full. Pick your level.

---

## What it is

You-Sir Juan™ ships as a complete hardware-software-service system in three tiers. Every tier runs the same backend, the same Associate Agent system, and the same private AI stack. What changes is the interface hardware and the AI runtime power.

---

## The three tiers

### Full — iPad Pro

The premium experience. A large-format iPad Pro mounted in the home — wall-mounted or countertop — running the full native iOS app with RealityKit 4 cinematic UI, Face ID biometric recognition, and Touch ID 2FA.

| Component | Spec |
|---|---|
| Interface | iPad Pro (any M-series, iPadOS 18+) |
| AI runtime | Mac mini M4 Pro 48GB (floor) → DGX Spark (ceiling) |
| Network | GL.iNet Flint 2 WireGuard gateway |
| Install | Field technician service included |

**Best for:** Family offices, high-touch households, anyone for whom the experience matters as much as the function.

---

### Standard — Raspberry Pi Kit

The accessible tier. A custom 3D-printed enclosure (Snapmaker 2.0) housing a Raspberry Pi 5 and an 8-inch touchscreen. Runs the web app as a kiosk — same design language, same Associate Agents, same backend. Bill of materials: approximately $150.

| Component | Spec |
|---|---|
| Interface | Raspberry Pi 5 + 8" touchscreen in Snapmaker enclosure |
| AI runtime | Mac mini M4 (entry) or any existing Mac |
| Network | GL.iNet router pre-loaded with WireGuard config |
| Install | Field technician service included |

**Best for:** Growing families, technically-capable households, anyone who wants the system without the iPad Pro price point.

---

### BYO — Web Only

No new hardware. The web app runs in any browser on any device the family already owns — laptop, tablet, existing iPad, phone. The Associate Agent system is identical. The cinematic 3D rendering is not available (browser limitation), but the full voice-first experience is.

| Component | Spec |
|---|---|
| Interface | Any browser (Chrome, Safari, Firefox) |
| AI runtime | Existing Mac or PC |
| Network | Existing home network (WireGuard optional) |
| Install | Self-service with documentation |

**Best for:** Early adopters who want to try the system before committing to hardware, households with an existing Mac mini they want to repurpose.

---

## Choosing a tier

| Question | Answer |
|---|---|
| Do you want the cinematic 3D experience? | Full |
| Do you want a dedicated kiosk without the iPad price? | Standard |
| Do you have a Mac and want to start today? | BYO |
| Will you eventually want the full experience? | Start BYO, upgrade to Full — same account, same memory, no migration |

---

## The installation kit (Full + Standard)

Every installed unit ships with:
- **GL.iNet Flint 2 router** pre-loaded with WireGuard config — creates the household's private AI mesh
- **Ansible playbook** that provisions the Mac mini runtime in under 60 minutes
- **Snapmaker 2.0-printed enclosure** (Standard SKU)
- **Pre-flashed router config** — WireGuard keys, firewall rules, device routing all set before delivery
- **Field technician install** — we come to the home, install, configure, and hand the family the keys
