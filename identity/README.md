# Full Identity Layer

## Purpose

The identity layer models relationships between:

- users
- households
- PMAs
- family offices
- organizations
- assistants
- trustees
- operators
- namespaces

| Entity | Purpose |
|---|---|
| User | individual operator |
| Workspace | isolated operational environment |
| Household | family continuity unit |
| PMA | member operations structure |
| Organization | enterprise coordination |
| Assistant | operational intelligence entity |
| Namespace | isolated memory boundary |
