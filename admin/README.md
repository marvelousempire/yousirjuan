# Admin Platform

# Purpose

The admin platform manages:

- customers
- assistants
- namespaces
- memory systems
- feature ledgers
- audit logs
- provisioning
- billing
- orchestration
- evaluations
- deployments

| Area | Purpose |
|---|---|
| Overview | platform health |
| Clients | customer workspaces |
| Assistants | assistant registry |
| Memory | namespace management |
| Features | feature ledger management |
| Pain Journal | user friction tracking |

| Evaluations | benchmark results |
| Infrastructure | nodes and routing |
