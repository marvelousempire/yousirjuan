# Ingestion Layer

## Purpose

The ingestion layer transforms raw information into structured full memory.

---

## Planned Workers

| Worker | Purpose |
|---|---|
| PDF Worker | parse and chunk documents |
| OCR Worker | scan image-based documents |
| Web Worker | ingest websites and structured web content |
| Transcript Worker | process audio and video transcripts |
| Embedding Worker | generate embeddings |
| Metadata Worker | enrich memory records |
| Queue Manager | coordinate ingestion jobs |

---

## Supported Sources

- PDFs
- images
- scanned records
- websites
- markdown
- Git repositories
- transcripts
- enterprise documents

---

## Long-Term Goal

Build multimodal full memory infrastructure.
